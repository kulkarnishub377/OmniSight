        Product Capabilities
        
Everything needed for a portfolio-grade AI platform

      

      

        
01
          
Incident Command Center

          
Filter, inspect, acknowledge, resolve, and investigate AI-generated incidents with evidence and actions.
          

        

        
02
          
Multimodal Event Operations

          
Vision, audio, sensor, webhook, and custom canonical event flows with Redpanda-ready topic contracts.

        

        
03
          
Documents & RAG

          
Upload manuals, chunk documents, index semantic memory, and retrieve evidence-backed chunks.

        

        
04
          
Agent Runtime

          
Observer, Retriever, Investigator, Planner, Safety Guard, and Executor workflow with trace viewer.

        

        
05
          
Knowledge Graph

          
Interactive graph of incidents, events, machines, zones, and documents with relationship inspection.

        

        
06
          
Governance & Audit

          
JWT, RBAC, audit logs, human approval policies, and guardrail decisions for enterprise AI safety.

Backend Domain Map
FastAPI
          

          

            
              

                Domain
                Files
                Purpose
              

            
            
              

                Auth/RBAC
                core/security.py · routers/auth.py · routers/users.py
                JWT login, password hashing, user roles, access control.
              

              

                Events
                routers/events.py · services/event_service.py
                Canonical event ingestion for cameras, sensors, webhooks, and services.
              

              

                Processing
                services/event_processor.py
                Anomaly derivation, time-window correlation, incident and alert creation.
              

              

                Incidents
                routers/incidents.py · models/models.py
                Incident queue, status updates, agent investigation triggers.
              

              

                Documents/RAG
                routers/documents.py · services/document_service.py · services/search_service.py
                Manual upload, chunking, deterministic semantic retrieval fallback.
              

              

                Agents
                services/agent_service.py
                Observer, Retriever, Investigator, Planner, Guard, Executor workflow.
              

              

                Actions/Audit
                routers/actions.py · routers/audit.py · services/action_service.py · services/audit_service.py
                Ticket stubs, notifications, audit logs, governance trail.
              

              

                Graph/System
                routers/knowledge.py · routers/system.py
                Knowledge graph, system stats, dashboard metrics.
